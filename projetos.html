Nome do projeto: Sistema de gerenciamento de tarefas

Resumo do projeto:

O sistema de gerenciamento de tarefas é uma aplicação web que permite aos usuários criar, visualizar e gerenciar tarefas em um ambiente online. A aplicação foi desenvolvida usando tecnologias web como HTML, CSS, JavaScript e o framework Ruby on Rails.

Funcionalidades:

• Cadastro de usuários: permite que os usuários se cadastrem na aplicação e criem suas contas pessoais para gerenciamento de tarefas;

• Login e logout: os usuários podem fazer login na aplicação usando suas credenciais e fazer logout quando necessário;

• Criação de tarefas: os usuários podem criar novas tarefas, atribuindo um título, uma descrição e uma data de prazo para conclusão;

• Visualização de tarefas: os usuários podem visualizar todas as tarefas criadas, incluindo suas informações detalhadas, como título, descrição, data de prazo e status atual;

• Edição e exclusão de tarefas: os usuários podem editar ou excluir tarefas existentes, alterando seu status, data de prazo ou excluindo completamente a tarefa;

• Priorização de tarefas: os usuários podem priorizar as tarefas com base em sua importância e urgência, para que possam trabalhar nas tarefas mais importantes primeiro;

• Notificações: os usuários recebem notificações por e-mail quando novas tarefas são criadas ou quando as tarefas estão próximas de sua data de prazo.

Resultados e impacto:

O sistema de gerenciamento de tarefas é uma ferramenta útil para usuários que precisam gerenciar várias tarefas ao mesmo tempo, permitindo que eles mantenham um controle eficiente de suas atividades e prazos. Além disso, o desenvolvimento deste projeto permitiu à estudante de ciência da computação adquirir experiência no desenvolvimento de aplicações web e em tecnologias como Ruby on Rails, além de melhorar suas habilidades em design de interfaces de usuário e gerenciamento de projetos de software.
